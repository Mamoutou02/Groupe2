Pour pouvoir crée compte l'utilisateur doit renseigner les champs suivants :
-Nom,
-Prénom,
-Email,
-Téléphone
-Nom d'utilisateur,
-Mot de passe.
Ensuite pour se connecter il aura besoin d'un : 
-Nom d'utilisateur, 
-Mot de passe.

Définition du Done (DoD) pour les fonctionnalités principales :

Pour chaque fonctionnalité principale, définir les critères de validation :
-Création  de compte Utilisateur : 
Saisir un nom(format lettre), un prénom(format lettre), un email, un numéro de téléphone, un nom d'utilisateur, un mot de passe(minimum 8 caractères), l’orientation vers la page d’accueil et en cas d’erreur l’apparition du message “Revoyez vos coordonnées”.
-Authentification de l’utilisateur : 
saisir son nom d'utilisateur et un mot de passe valide, redirection vers la page d’accueil avec l’apparition du message “Bienvenue” et si les données saisies sont incorrectes l’apparition d’un message d’erreur ”Donnée incorrect”.
