En tant qu'utilisateur je veux une interface utilisateur simple coherent etattirant 
1. Conformité aux maquettes
Les conceptions respectent les maquettes approuvées et les spécifications de design.
2. Tests de convivialité
Réalisation de tests de convivialité avec des utilisateurs pour s'assurer que l'interface est intuitive et facile à utiliser.
3. Accessibilité
L'application respecte les normes d'accessibilité (par exemple, WCAG 2.1) pour s'assurer que tous les utilisateurs, y compris ceux ayant des handicaps, peuvent naviguer.
4. Responsive Design
L'interface fonctionne correctement sur une variété d'appareils et de résolutions d'écran (mobiles, tablettes, ordinateurs de bureau).
5. Retours utilisateurs intégrés
Les retours des utilisateurs sont pris en compte et les ajustements nécessaires ont été faits avant la finalisation.
6. Documentation
La documentation des composants UI/UX est mise à jour, y compris les styles, les comportements et les éléments interactifs.
7. Tests fonctionnels et de régression
Tests réalisés pour vérifier que les nouvelles fonctionnalités n'affectent pas le fonctionnement existant.
8. Performance
Les aspects de performance UI sont évalués (temps de chargement, réactivité des éléments interactifs).
9. Validation avec stakeholders
Validation et acceptation de l'interface par les parties prenantes concernées avant le déploiement.
10. Déploiement en environnement de test
L'interface est déployée et testée dans un environnement de test pour vérification avant la mise en production

Définition du Done (DoD) pour les fonctionnalités principales :

Pour chaque fonctionnalité principale, définir les critères de validation :
-Création  de compte Utilisateur : 
Saisir un nom(format lettre), un prénom(format lettre), un email, un numéro de téléphone, un nom d'utilisateur, un mot de passe(minimum 8 caractères), l’orientation vers la page d’accueil et en cas d’erreur l’apparition du message “Revoyez vos coordonnées”.
-Authentification de l’utilisateur : 
saisir son nom d'utilisateur et un mot de passe valide, redirection vers la page d’accueil avec l’apparition du message “Bienvenue” et si les données saisies sont incorrectes l’apparition d’un message d’erreur ”Donnée incorrect”.
